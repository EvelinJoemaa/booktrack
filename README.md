# booktrack

This is a simple booktracking app with VUE and Express. It uses plain json array to store notes.

## Project management

[https://www.pivotaltracker.com/n/projects/2633250](https://www.pivotaltracker.com/n/projects/2633250)

## Project setup

```
cp .env.example .env
```

```
npm install
```

### Running the app

```
npm start
```
